What is Sanitarium.io?

Welcome to Sanitarium.io, a facility for holding the criminally insane. Our archive exposes a JSON API, enabling you to query hand-curated data on horror movie antagonists.

Patient records contain an in-depth profile and previous criminal activity, including a list of known victims. Our facility basement is used to house our aquatic killers.

API Usage

This is a consumption-only API - only the HTTP GET method is available on resources. No authentication is required.

Retrieve a random killer in JSON format.

GET https://api.sanitarium.io/killers/random

Example response:

Method Path Description
GET /killers/random Find random killer.
GET /killers/{~slug} Find killer by slug, note the tilde-prefix (~).
GET /classifications List all killer classifications.

Query parameters are all optional and may be appended to the /killers/random endpoint, to influence the response. 

GET https://api.sanitarium.io/killers/random?weapon=teeth&residence=maine
Parameter Type Description
name string Fuzzy search on both killers name and known aliases.
classification string Fuzzy search on killers classification.
residence string Fuzzy search on killers primary residence.
weapon string Fuzzy search on killers primary weapon of choice.
Fair Use Policy

Sanitarium.io is free and open to use. It is also popular. Because of this, we ask every developer to abide by our fair use policy. People not complying with the fair use policy will have their IP address permanently banned.

We will not tolerate denial of service attacks.

Rules:

  • You can make up to 200 requests per hour.
  • Locally cache resources whenever you request them.
  • Use the correct user-agent header in API requests.
  • Be helpful and friendly to your fellow Sanitarium.io developers :-).

Privacy

Sanitarium.io was a weekend project and is hosted on Heroku. All traffic is encrypted by Cloudflare which enables SSL/TLS. IP addresses are captured solely for the rate limiter. No data will ever be shared, except for internal usage for anonymous statistics.
